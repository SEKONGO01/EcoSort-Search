# ♻️ EcoSort-Search

Application web qui aide au tri sélectif : l'utilisateur recherche un produit,
l'app le trouve sur Jumia, puis une IA (Deep Learning) prédit la bonne
poubelle de tri (Jaune, Verte, Bleue, D3E, Marron/Noire).
